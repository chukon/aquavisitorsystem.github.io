Aqua Visitor System

The Aqua Visitor System is comprised of a web-based scheduling app, iPad app, and name badge printer that allows Aqua employees to schedule and visitors to check-in and check-out with a touchless experience.

View/Edit Visitor Schedule

back to top

2.    Enter in Keyword or [Aqua Network Login ID] (example: ckonkol) (see #1 in image below)

c.     all = all active schedules

d.    inactive = all inactive schedules

e.    [Aqua Network Login ID] = all active schedules for Aqua employee

5.    Edit fields.

a.    All fields are editable except for Aqua Employee Email (see #1 in image below)

b.    If you update field data, click Update Visit button to update schedule (see #2 in image below)

6.    Email updated schedule to visitor by clicking link Click here to create email to guest (see #3 in image below)

Update Visitor Schedule: Active/In-Active

back to top

By default, all schedules are active. However, there may be a schedule that is on hold or cancelled all together. In those cases, you need to make schedule In-Active.

1.    Go to aquavisitorsystem.github.io & click on View Scheduled Visits (see #1 in image below)

2.    Enter in Keyword or [Aqua Network Login ID] (example: ckonkol) (see image below)

a.    today = schedule for todays date

b.    date = schedule for a specific date

c.     all = all active schedules

d.    inactive = all inactive schedules

e.    [Aqua Network Login ID] = all active schedules for Aqua employee

3.    Click button View Scheduled Visits (see image below)



4.    Select schedule to make in-active/active by clicking the Click here link. (see #1 in image below)

5.    Click the link below Status to change status to active or in-active (see #1 in image below)

Reset Check-in/Check-out data for Visitor

back to top

There might be times the check-in/check-out data needs reset. For example, if the printer fails to print a name badge due to being out of paper or power issue.

1.       Go to aquavisitorsystem.github.io & click on View Scheduled Visits (see #1 in image below)

2.       Enter keyword all and press [enter] key

3.       Select guest that needs check-in/check-out data reset (click the click here link)

4.    Click the Click here to reset check-in/check-out data link to reset check-in/check-out data

Making sure system is ready to use (IT Use Only)

back to top

 

1)      Power

a)       Make sure power strip is plugged securely into the wall

b)      Make sure iPad & Printer are plugged into power strip located behind iPad kiosk

2)      Make sure Wi-Fi is connected to correct printer Wi-Fi DIRECT-06442_QL-1110NWB

a)       Settings > Wifi > DIRECT-06442_QL-1110NWB

3)      Make sure AquaCheckIn app is open




4)      Make sure Guided Access is started. This helps app stay open and disallows other iPad functions

a)       Triple tap top left button to start guided access



b)      To exit Guided Access triple tap top left button and use code 6306

5)      If iPad goes to sleep mode and is locked you can unlock using iPad lock code 6306
