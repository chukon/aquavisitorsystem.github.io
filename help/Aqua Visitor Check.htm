Aqua Visitor System

The Aqua Visitor System (also called Visitor Management System or VMS) is comprised of a web-based scheduling app, iPad app, and name badge printer.

Visitors receive a generated name badge with QR code that allows them to check-in and check-out with a touchless experience.

Aqua employees schedule, send email invitations, receive email notifications when their visitor arrives and can view multiple reports!

Update Visitor Schedule: Active/In-Active

back to top

By default, all schedules are active. However, there may be a schedule that is on hold or cancelled all together. In those cases, you need to make schedule In-Active.

1.    Go to aquavisitorsystem.github.io & click on View Scheduled Visits (see image below)

2.    Enter [Aqua User ID] > [Enter] key. (example: ckonkol) - (see image example below)

3.    Select schedule to make in-active/active by clicking the Click here link. (see image below)

4.    Click the link below Status to change status to active or in-active (see image below)

Reports

back to top

 

1.    Go to aquavisitorsystem.github.io & click on View Scheduled Visits (see #1 in image below)

2.       Enter [Aqua User ID] > [Enter] key OR select below keywords from dropdown.

Keywords to use (keywords in bold below):

a.       [Aqua User ID] = shows active schedules for [aqua user id]. Example: ckonkol

b.       today = show today schedule

c.       myvms = shows active schedules for [aqua user id]. Example: ckonkol

d.       activitylog = show live visitor log

e.       active = shows all schedules greater or equal to todays date

f.        name = shows all active schedules by last name

g.       date = shows schedules for a specific date

h.       all = shows all schedules

i.         inactive = show all inactive schedules

j.         count = show visitor count from date

k.       loguserid = get report of guest check-in/check-out sessions for an Aqua Employee UserID. Example: ckonkol

l.         logname = get report of check-in/check-out sessions for a guest

m.     logall = get report of all check-in/check-out sessions

n.       logtoday = get report of check-in/check-out sessions for today

o.       logweek = get report of all check-in/check-out sessions for 7 days

p.       logdate = get report of check-in/check-out sessions by date

q.       printjobs = get report of how many labels have been used and how many labels remain.

Reset Check-in/Check-out data for Visitor

back to top

There might be times the check-in/check-out data needs reset. For example, if the printer fails to print a name badge due to being out of paper or power issue.

1.       Go to aquavisitorsystem.github.io & click on View Scheduled Visits (see #1 in image below)

2.       Enter keyword today and press [enter] key

3.       Select guest that needs check-in/check-out data reset (click the click here link)

4.    Click the Click here to reset check-in/check-out data link to reset check-in/check-out data

Making sure system is ready to use

back to top

1)      Power

a)       Make sure power strip is plugged securely into the wall

b)      Make sure iPad & Printer are plugged into power strip located behind iPad kiosk

2)      Make sure Wi-Fi is connected to correct printer Wi-Fi DIRECT-06442_QL-1110NWB

a)       Settings > Wifi > DIRECT-06442_QL-1110NWB

3)      Make sure AquaCheckIn app is open




4)      Make sure Guided Access is started. This helps app stay open and disallows other iPad functions

a)       Triple tap top left button to start guided access



b)      To exit Guided Access triple tap top left button and use code 6306

5)      If iPad goes to sleep mode and is locked you can unlock using iPad lock code 6306
